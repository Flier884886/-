# -宿舍纪律扣分防治项目
    晚上宿管一直严查纪律，宿舍说话则扣分，探头查看宿管动向也扣分，于是便出现了此项目。本项目基于ESP8266模块和人体存在传感器，通过mesh组网技术将宿管的动向广播给各个终端，达到提前预警的效果。
    天下苦宿管久矣！之前技术不够，宿舍一直讲话常被扣分，甚至几近达到接受处罚的界限。现在技术足够了，于是开发了这个项目。本项目的核心为一块ESP8266模块，通过mesh组网，理论上能把整栋宿舍悄咪咪地打造成一张天罗地网，宿管动向运之于掌上，再也不用担心说话被发现进而扣分了。传感端连接人体存在传感器，传感端将信息包装后在网络中广播，所有终端都可以接收到宿管的位置，宿管接近就亮灯。使用者发现亮灯后，距离宿管的到达还有一定的时间，完全足够shutdown来暂避风头了。
    条件不足者接收器单板便可以接收信息（即板载LED灯），条件优越者可以精确到宿管究竟从哪边靠近。本项目保留了较多的改进空间，如果加入显示屏，那么就真的是天罗地网了，包括宿管在那一层楼的哪个位置，甚至也能知道隔壁栋的宿管在哪里。
